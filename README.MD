## Font

- Font type
[Poppins on Google Fonts](https://fonts.google.com/specimen/Poppins?query=pop)

- Font color -Default ** NOT BLACK **
```css
color: #2D3748;
```
- Font Size - Header
```css
font-size: 0.9rem;
```
- Font Size - Default
```css
font-size: 0.8rem;
```

## Styles

```css
	margin-top: 0rem;
	width: 100%;
	height: 2rem;
	text-align: center;
	border-radius: 0.75rem;
	padding: 1rem 1;
	background-color: white;
	border-color: white;
	font-size: 0.7rem;
}
```

- Border-radiue
```css

```

## Colors

- Font (Default) - dark blue
```css

```
- Background (Default) - light gray

